## 💻 Projects & Experience

### ScopeCLIP: Region-Mask Training-Free Open-Vocabulary Semantic Segmentation with Lightweight SAM2

Capstone Design Project, Sungkyunkwan University  
Advisor: Prof. Jongpil Jeong  
Industry Partner: Duplex  

This project focuses on training-free Open-Vocabulary Semantic Segmentation (OVSS) using CLIP and SAM2 for smart factory vision inspection. ScopeCLIP integrates region-based attention restriction, Scope Reconstruction (SR), and region-level label refinement, Map Correction (MC), into a NACLIP-based segmentation pipeline.

The main objective was to analyze whether a lightweight SAM2 backbone, Hiera-Tiny, can replace the high-cost Hiera-L backbone while preserving segmentation accuracy and reducing computational cost.
